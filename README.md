# VGameKit

VGameKit is a modular Unity framework designed to streamline game development by providing a collection of pre-built tools and systems.

## Modules

### VGameKit

The core module of the framework, VGameKit offers a wide range of essential functionalities:

- **App**: Manages the application lifecycle and global state.
- **Config**: Handles game configuration and settings.
- **Core**: Provides fundamental classes and utilities.
- **Log**: A simple and extensible logging system.
- **ProcessFlows**: Manages game state transitions and complex sequences.
- **Spawner**: A system for efficiently spawning and despawning game objects.
- **UI**: Includes a robust UI management system with support for menus and events.
- **Utilities**: A collection of helper classes and extension methods to simplify common tasks.

```js
"com.cngz.vgamekit": "https://github.com/CengizKuscu/VGameKit.git?path=Assets/VGameKit#v0.0.4"
```

### VGameKit.GA

This module integrates Google Analytics into your project, allowing you to track player behavior and game events.

- **GA_Initialization**: A simple script to initialize the Google Analytics SDK.

### VGameKit.GoogleAds

A comprehensive solution for integrating Google Mobile Ads into your game.

- **Ad Types**: Supports Banner, Interstitial, and Rewarded ads.
- **Consent Management**: Manages user consent for personalized advertising in compliance with GDPR and other privacy regulations.
- **Editor Extensions**: Provides a user-friendly interface in the Unity Editor to configure ad units and settings.
- **Privacy Button**: A pre-built UI component that allows players to manage their privacy preferences.
